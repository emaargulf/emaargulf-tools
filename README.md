# Emaar Gulf Engineering Consultants - Internal Tools Portal

This repository hosts the official internal web applications, automation scripts, and employee utilities for Emaar Gulf Engineering Consultants.

## 🚀 Live Applications
* **Email Signature Generator:** https://emaargulf.github.io/emaargulf-tools/signature/signature_generator.html
* **Eid Greetings App:** https://emaargulf.github.io/emaargulf-tools/eid/

## 📂 Repository Structure
* `/signature/`: Dedicated application module for the staff email signature generator.
* `/eid/`: Dedicated application module for Eid greetings and related assets.

## 🔐 Access Control
* **User Access:** Required for general staff to generate signatures (protected by client-side access code).
* **Admin Access:** Required for IT/Management to update company-wide links or tool logic.
